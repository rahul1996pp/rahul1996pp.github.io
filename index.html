Go · single binary

netmon — Network Monitor & Firewall

ProblemOperators need per-process network visibility and host firewalling without installing heavy agents or shipping traffic to a cloud service.
BuiltA single Go binary that captures packets, attributes every flow to its owning process, enriches IPs with GeoIP, renders a D3 force-directed network map, and runs a block/alert rule engine — all behind a live web UI reachable from any LAN device.
ResultZero-dependency, self-hosted monitoring on Linux, macOS and Windows: real-time bandwidth, hundreds of live flows, per-process usage, domains, alerts and on-demand PCAP capture.
